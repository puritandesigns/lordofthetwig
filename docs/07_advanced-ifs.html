More If Statement Possibilities

Combining if statements with ands and ors 


{% if location == 'weathertop' and ring_wraiths > 5 %}
    Frodo will be captured.
{% else %}
    Frodo will escape.
{% endif %}

{% set gandalf = {
    job: 'Wizard',
    friends: {
        count: 8,
        status: 'in mortal danger'
    }
} %}
{% if location == 'moria' or gandalf.friends.status == 'in mortal danger' %}
    Gandalf will sacrifice himself.
{% endif %}

Frodo will escape.

Gandalf will sacrifice himself.

Working with strings (length, starts/ends with)


{% set hobbit = 'frodo' %}
{% if hobbit|length == 5 and (hobbit starts with 'f' or hobbit ends with 'o') %}
    The Hobbit is Frodo
{% endif %}

The Hobbit is Frodo

Working with numbers (odd/even and divisble by)


{% set fellowship = 9 %}
{% if fellowship|length == 9 or fellowship is odd %}
    
Boromir has not fallen yet.

{% elseif fellowship is odd %}
    Alas for Boromir the Brave!
{% endif %}

Boromir has not fallen yet.

Working with arrays (iterable, length, in)


{% set fellowship = [
    'frodo',
    'sam',
    'merry',
    'pippin',
    'aragorn',
    'gandalf',
    'boromir',
    'legolas',
    'gimli'
] %}
{# if not sure if fellowship is an array or number... #}
{% if fellowship is iterable and (fellowship|length == 9 or 'boromir' in fellowship) %}
    
Boromir has not fallen yet.

{% endif %}

Boromir has not fallen yet.

A mortal... who keeps one of the Great Rings, does not die, but he does not grow or obtain more life, he merely continues, until at last every minute is a weariness. And if he often uses the Ring to make himself invisible, he fades: he becomes in the end invisible permanently, and walks in the twilight under the eye of the dark power that rules the Rings.
