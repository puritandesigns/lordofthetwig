For Statements

The use of {% for %} loops allows you to go through arrays or objects one item at a time.

<ul>
{% for member in fellowship %}
    <li>
        {{ member.full_name -}}

    </li>
{% endfor %}
</ul>

For key and value


{% for name, member in fellowship %}
<li>
    <h4>{{ name|title }}</h4>
    {% for key, value in member %}
        {{ key }}:
        {% if value is iterable %}
            {% for val in value %}
                {{ val }},
            {% endfor %}
        {% elseif value is empty %}
            nope...
        {% else %}
            {{ value }}
        {% endif %}
        <br>
    {% endfor %}
</li>
{% endfor %}
